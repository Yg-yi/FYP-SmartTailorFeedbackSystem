# SmartTailor: Customer - Tailor Feedback & Rating Module

A bespoke digital tailoring platform and feedback management system designed to bridge the communication and quality-assurance gap between custom tailors and clients.

---

## 📌 Project Overview
* **Domain**: E-Commerce & Service Matching Platform (Final Year Project)
* **Core Objective**: Streamline custom clothing orders, provide asynchronous local telemetry logging, and ensure review reliability through automated text moderation.
* **Tech Stack**: PHP, MySQL, Tailwind CSS, Agile Methodology

---

## 🚀 Key Features & Architecture
1. **Bespoke Tailoring Matching Engine**: Algorithmic pairing of user preferences with specialized tailors.
2. **Review & Rating Module**: Structured feedback loops allowing clients to rate craftsmanship, delivery time, and service quality.
3. **Automated Text Verification**: Built-in moderation mechanism to screen review inputs and filter inappropriate or invalid feedback.
4. **Asynchronous Local Telemetry Logs**: Non-blocking local logging to track user interactions and system health metrics.

---

## ⚙️ Tech Stack & Tools
* **Frontend**: Tailwind CSS, HTML5, JavaScript
* **Backend**: PHP (Server-side logic and API endpoints)
* **Database**: MySQL (Relational schema for users, orders, and ratings)
* **Development Methodology**: Agile (Sprint-based feature delivery)

## ⚙️ How to Set Up Locally
1. **Clone the repository:**
```bash
git clone https://github.com/Yg-yi/FYP-SmartTailorFeedbackSystem.git

2. **Navigate to the project directory:**
```bash
cd FYP-SmartTailorFeedbackSystem

3. **Import Database:**
•Set up a local server environment (e.g., XAMPP / WampServer).

4. **Configure Connection:**
•Update your database credentials in the backend connection configuration file.

•Move the project folder to your local server root (e.g., htdocs for XAMPP) and launch it via your browser.
